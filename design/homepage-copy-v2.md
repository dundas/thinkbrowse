## Section: How it works

> Merges Local/Cloud into one "flexible" message. Elevates Record Mode.

### Give it a command
Describe what you need in plain English — from the web dashboard, CLI, or API. The agent plans the steps and starts browsing.

### It works, you don't
The agent navigates, clicks, fills forms, and extracts data. Watch in real-time or walk away. If it hits MFA or a judgment call, it pauses and asks you.

### Get back results
Screenshots, structured data, downloaded files — delivered when the task is done. Review the full action history or just grab the output.

## Section: Teach Once, Run Forever (Record Mode — elevated)

> Previously buried. Now its own section.

**Headline:**
Teach once. Run forever.

**Body:**
Record yourself doing a task — clicks, keystrokes, and your spoken intent. ThinkBrowse doesn't just replay the macro. It learns the workflow, so it handles variations, layout changes, and edge cases on its own.

**CLI example:**
```bash
thinkbrowse record start
# do the task in your browser
thinkbrowse record stop
thinkbrowse replay "last" --adapt
```
